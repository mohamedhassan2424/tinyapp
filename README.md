## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

1.)  Install all dependencies (using the `npm install` command)
2.) Run the development web server using the `node express_server.js` command.
3.) Run localhost http://localhost:8080/ to connect
